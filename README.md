## 🎥 What is Remotion?  
**Remotion** is the ultimate tool to create **videos programmatically** using **React**! Build, animate, and render stunning videos with full control over components.  

---

## ⚡ Quick Start  

### 🔧 Install Dependencies  
```sh
npm install
```

### 🎬 Start Preview Mode  
```sh
npm start
```

### 🏗️ Render Your Video  
```sh
npm run build
```

### 📦 Upgrade Remotion  
```sh
npm run upgrade
```
